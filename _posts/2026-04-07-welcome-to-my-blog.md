## Why I started this blog

I wanted a clean and free way to share what I learn about **data science**, **AI**, and **cloud**.  
GitHub Pages with Jekyll is a great choice because it is:

- Free to host
- Easy to maintain
- Friendly for Markdown writing
- Simple to version control with GitHub

## What you can expect here

This blog will include posts on:

- Data science concepts and hands-on notes
- AI tools and experiments
- Cloud projects and deployment tips

## Why I chose Minimal Mistakes

The **Minimal Mistakes** theme gives me:

- A professional layout
- Built-in support for categories and tags
- Read time and author profile features
- A structure that is easy to extend later

## Writing workflow

Each post is just a Markdown file inside the `_posts` folder.  
That means I can focus on writing content instead of managing a complex website setup.

## Next steps

My next posts will cover practical topics, small experiments, and step-by-step guides.

Thanks for reading.
